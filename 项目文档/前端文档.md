# 前端技术文档

本项目前端基于现代化的 Web 开发栈，采用单页应用 (SPA) 架构，深度集成响应式设计与角色权限控制。

## 🏗️ 架构概览

- **框架**: [React 18](https://reactjs.org/)
- **构建工具**: [Vite](https://vitejs.dev/)
- **样式**: [Tailwind CSS](https://tailwindcss.com/)
- **路由**: [React Router v6](https://reactrouter.com/)
- **图标**: [Heroicons](https://heroicons.com/)
- **图表**: [Recharts](https://recharts.org/)

## 🔑 核心功能实现

### 1. 路由与权限控制 (`App.jsx`)
系统采用基于角色的访问控制 (RBAC)。在 `App.jsx` 中定义了 `ProtectedRoute` 组件：
- **逻辑**：检查 `localStorage` 中的用户角色。
- **限制**：`UserManagement` 和 `SystemMaintenance` 仅限 `admin` 角色访问。
- **持久化**：登录状态及用户信息通过 `localStorage` 在会话间保持。

### 2. 交互式仪表盘 (`pages/Dashboard.jsx`)
- **指标统计**：通过 API 获取后端聚合数据（总库存、月销售额等）。
- **趋势分析**：利用 Recharts 渲染最近 7 天的销售趋势图，数据后端来源于 `sp_sales_trend` 存储过程。

### 3. 数据管理页面
每个数据管理模块（药品、客户、供应商、订单）均具备以下特性：
- **实时搜索**：输入关键字时自动触发后端模糊查询存储过程。
- **分页处理**：自定义 `Pagination` 组件与后端 `LIMIT/OFFSET` 逻辑完美衔接。
- **反馈系统**：集成全局 `Toast` 组件，实时提示操作结果（成功/失败）。

## 📂 目录结构说明

- `src/components/`：可复用的 UI 单元（侧边栏、导航栏、弹窗、分页、加载态）。
- `src/pages/`：业务页面逻辑。
- `src/index.css`：核心设计系统，预设了翡翠绿 (Emerald) 主体色调。
- `src/App.jsx`：应用的根入口及全局状态、路由分发中心。

## 💡 开发习惯
- **组件化**：保持小而美的组件拆分原则。
- **响应式**：优先移动端适配，侧边栏在小屏下自动折叠为悬浮菜单。
